# Personal Portfolio

A modern, responsive personal portfolio built with [Next.js](https://nextjs.org/) and [Tailwind CSS](https://tailwindcss.com/).

## 🚀 Features

- **Modern Tech Stack**: Built with Next.js 16, React 19, and Tailwind CSS 4.
- **Responsive Design**: Fully responsive layout that looks great on all devices.
- **Component-Driven Architecture**: Modular components for sections like Hero, About, Skills, Projects, Tools, and Contact.
- **Type-Safe**: Written in TypeScript for better maintainability and developer experience.

## 📁 Project Structure

The main application code is located in the `app` directory:

- `app/component/`: Contains all reusable UI components.
  - `hero/` - Landing section.
  - `about/` - Personal introduction.
  - `skills/` - Technical skills showcase.
  - `projects/` - Featured projects gallery.
  - `tools/` - Development tools and setup.
  - `contacts/` - Contact information and forms.
  - `navbar/` & `footer/` - Navigation and footer.

## 🛠️ Getting Started

First, install the dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
